<h1>Password Strength Classification – Ensemble Learning</h1>

<h2>📌 Project Overview</h2>
<p>
A fictional cybersecurity team wanted to reduce the number of weak passwords used across their 
organization. Weak passwords were leading to frequent security breaches and compliance issues. 
They needed a machine learning model that could automatically classify password strength and 
guide users to create more secure credentials.
</p>

<h2>🎯 Business Problem</h2>
<p>
Employees often created predictable or weak passwords, increasing vulnerability to attacks. 
Manual audits were slow and ineffective, and the company needed an automated solution.
</p>

<h2>🧠 What This Project Delivers</h2>
<ul>
  <li>Built an ensemble‑based classifier to categorize passwords as Weak, Medium, or Strong.</li>
  <li>Used NLP techniques to extract meaningful features from text.</li>
  <li>Improved security posture by identifying risky passwords early.</li>
  <li>Provided insights into common weak‑password patterns.</li>
</ul>

<h2>🛠️ Techniques Used</h2>
<ul>
  <li>Ensemble Learning (Random Forest, Bagging)</li>
  <li>NLP Feature Extraction (TF‑IDF, Character Patterns)</li>
  <li>Text Preprocessing</li>
  <li>Model Evaluation (Accuracy, Confusion Matrix)</li>
</ul>
